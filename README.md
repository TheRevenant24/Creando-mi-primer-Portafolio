# Mi Portafolio Flex

Este proyecto es un portafolio personal desarrollado como parte del programa **Análisis y Desarrollo de Software (ADSO)**.  
El objetivo es demostrar dominio del modelo de caja flexible (**CSS Flexbox**) aplicando maquetación moderna y responsive.

---

## Objetivos
- Aplicar correctamente las propiedades de Flexbox en todas las secciones.
- Construir un diseño **responsive** con media queries para tablet y móvil.
- Implementar interactividad básica con **JavaScript** (menú hamburguesa).
- Publicar el proyecto en un repositorio público de **GitHub** y desplegarlo con **GitHub Pages**.

---

## Tecnologías utilizadas
- **HTML5**: estructura semántica (header, nav, section, footer).
- **CSS3 (Flexbox)**: maquetación flexible y adaptable.
- **JavaScript**: interactividad del menú hamburguesa.
- **Git & GitHub**: control de versiones y publicación del proyecto.

---

## Criterios cumplidos
- Uso de propiedades Flexbox: `display: flex`, `flex-direction`, `justify-content`, `align-items`, `flex-wrap`, `gap`, `flex`, `order`.
- Secciones obligatorias: **navegación, hero, sobre mí, galería, footer**.
- Diseño responsive con **dos media queries**.
- Interactividad con **JavaScript**.
- Repositorio con commits descriptivos y README.
